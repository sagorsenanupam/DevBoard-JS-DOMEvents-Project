Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the document object model. Given below:

  1. document.getElementById("id") : Selects element by ID
  2. document.getElementsByClassName("class") : Selects elements by class
  3. document.getElementsByTagName("tag") : Selects elements by tag name
  4. document.querySelector("selector") : Selects the first match using CSS selector
  5. document.querySelectorAll("selector") : Selects all matches using CSS selector

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns HTML content including tags where innerText returns the user interface visible texts. Lastly the textContent returns all the text content including the hidden text also. it does not show us the HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation means adding the event to the parent element so that when we click the child element it goes to their parent element. So we do not need to write in every child to do that.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the clicking any element it bubble up gradually to their parent element in the DOM

Question-5: How do you create, add, and remove elements using JavaScript?

1. document.createElement('tag') : creating
2. parent.appendChild(newElement): adding
3. parent.removeChild(childElement) : removing
